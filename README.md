# 🌾 Crop Deal

Crop Deal is a full-stack web application designed to simplify agricultural operations by providing a platform for managing crop-related activities. The application offers a responsive user interface and a secure backend to streamline crop management, user interactions, and data processing.

## 🚀 Features

- User Authentication and Authorization
- Crop Listing and Management
- Order and Transaction Management
- Responsive User Interface
- RESTful API Integration
- Secure Data Management
- CRUD Operations
- Role-Based Access Control
- Optimized Database Operations

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/CropDeal.git
```

### Backend

```bash
cd Backend
```

- Update the SQL Server connection string in **appsettings.json**
- Run database migrations

```bash
dotnet ef database update
```

Run the API

```bash
dotnet run
```

### Frontend

```bash
cd Frontend
npm install
ng serve
```

Open:

```
http://localhost:4200
```
